# Daraz Playwright Automation Framework

This is a Playwright-based automation framework for testing the Daraz e-commerce application. 
It is built with JavaScript and follows the Page Object Model (POM) design pattern.

## Tech Stack

- Node.js
- Playwright
- JavaScript
- VS Code (recommended)
- GitHub Actions (for CI/CD)

## Project Structure
├── Pages/ # Page objects, utilities, fixtures
├── tests/ # Test scripts
├── playwright.config.js
├── package.json
├── README.md
└── .gitignore
